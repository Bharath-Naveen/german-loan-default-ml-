# 💳 Loan Default Prediction using Machine Learning

This project analyzes financial and demographic data to predict loan default risk using multiple machine learning models.

---

## 🚀 Project Overview

Financial institutions face significant challenges in identifying high-risk borrowers.

This project:
- Analyzes borrower data
- Identifies key default risk factors
- Builds predictive models to classify defaulters

---

## 📊 Key Highlights

- 📈 Full ML pipeline: EDA → preprocessing → modeling → evaluation
- ⚙️ Compared multiple models:
  - Logistic Regression
  - Decision Tree
  - Random Forest
  - SVM
  - Gradient Boosting
- 🏆 Achieved best performance using **Gradient Boosting**
- 🎯 Focused on **recall** (important for identifying defaulters)

---

## 🔍 Key Insights

- Low checking balance → higher default risk
- Short employment duration → higher risk
- Poor credit history → strong predictor
- High loan-to-income ratio → increased default probability
